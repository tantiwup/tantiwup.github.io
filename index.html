Projects


3D Printed Hand

A glove with strategically placed flex sensors keeps track of the user's hand movements. A simple Python application then saves the movements to be played back at any time to a mechanical 3D printed hand of our own design.


Technologies

Python, Arduino, SolidWorks, 3D Printing


Future Goals

Improve the accuracy of the recordings and rewrite the control application for Android/IoS.


California Wildfire Simulator

This application uses U.S. Forest Service wildfire data to generate Monte Carlo simulations that predict the likelihood of wildfire at the user's chosen location(CA only).

View Demo


Technologies

R, Shiny, Leaflet for R, shinyapps.io


Future Goals

Factor real-time weather data into the algorithm and expand support to the continental United States.


CrowdFoods

If you've ever tried to plan a party or feed a crowd, you know that it requires a lot of planning and organization. This web app makes it easy to pay for food for your entire group. This app won the Postmates API award at LA Hacks 2016.
